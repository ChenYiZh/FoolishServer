## 什么是Foolish Server？
欢迎使用 Foolish Server，这是一款免费开源的游戏服务器，可使用Http，WebSocket和Socket协议通讯，基于 .Net Standard 2.0 开发，所以 .Net Core 支持的平台，它都可以支持。客户端可以使用Unity，Unreal（作者现阶段只在这两个平台开发）。简单快捷搭建部署游戏。
